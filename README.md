# linux-dotfiles [![Build Status](https://travis-ci.org/Globidev/linux-dotfiles.svg?branch=master)](https://travis-ci.org/Globidev/linux-dotfiles)
My dotfiles for a linux environment

### Usage
```sh
sudo \
    DOT=$PWD \
    USER=target_user \
    HOME=target_users_home \
    ./scripts/setup.sh
```
